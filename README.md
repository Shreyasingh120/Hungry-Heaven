# HungryHaven
About the Project

Hungry Heaven is a simple food delivery website built using HTML, CSS, and JSON. It allows users to browse a variety of food items, add them to their cart, and place orders seamlessly.

Features

User-friendly Interface: A visually appealing and responsive design.

Food Catalog: Displays a variety of food items with descriptions and prices.

Cart System: Users can add or remove items from the cart.

JSON-based Data Management: Food item details are stored in a JSON file.

Responsive Design: Works on both desktop and mobile devices.
